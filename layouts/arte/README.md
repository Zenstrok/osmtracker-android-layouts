## Disposición de Arte

Esta disposición surge como iniciativa para facilitar el mapeo del transporte público en distintas zonas del país.Implementando el layout se podrá mapear los tres tipos de paradas, lugares de estacionamiento de taxis, trenes y bicicletas, además, de ciclovías y lugares de alquiler de bicicletas.
